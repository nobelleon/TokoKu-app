## Features

- Get Firebase authentication from console.firebase.google.com  API
- Get google map from cloud.google.com  API
- providers
- Call Demo
- Video Call Demo

## Plugins :

```bash
  provider: ^4.0.0 
  intl: ^0.15.8  
  http: ^0.12.1  
  shared_preferences: ^0.5.3+1
```
